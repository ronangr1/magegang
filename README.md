# Magegang Project

Member of the .phtml gang ONLY!

## How to become a member

1. Find a nickname
2. Fork the project
3. Add your profile in the `gang.json` file (example below)
4. Submit a PR
5. Wait for approval

This is what the file format should look like:

```
{
  "members": [
    {
      "name": "Your Name",
      "nickname": "Fresh Install",
      "company": "Your Company Name",
      "title": "Your Title"
    }
  ]
}
```
